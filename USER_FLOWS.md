# Content Creator User Flows

## Flow 1: The "Profit-Path" (Upload to Distribution)
1.  **Trigger:** User logs in and lands on the Dashboard.
2.  **Action:** User drags 100 images into the "AI Studio" upload zone.
3.  **System:** AI begins scanning (Computer Vision).
4.  **Action:** User reviews the AI-generated titles and keywords in "Bulk Edit" mode.
5.  **System:** System flags 2 images with "Potential Trademark Logo Detected."
6.  **Action:** User removes the 2 flagged images and clicks "Distribute."
7.  **System:** Displays a checklist of connected agencies (Shutterstock, Adobe, Getty).
8.  **Action:** User selects all and clicks "Finalize Sync."
9.  **Outcome:** Assets are pushed to agency queues; User receives a success notification.

## Flow 2: Market-Driven Creation
1.  **Trigger:** User checks the "Market Intelligence" tab.
2.  **System:** Displays "Trending Now: Sustainable Energy in Southeast Asia."
3.  **Action:** User views specific top-performing keywords for this trend.
4.  **Action:** User plans a photoshoot based on this data.
5.  **Outcome:** User increases the likelihood of sales by producing high-demand content.

## Flow 3: Agency Integration Onboarding
1.  **Trigger:** New user completes registration.
2.  **Action:** User is prompted to "Connect Your First Agency."
3.  **Action:** User selects "Adobe Stock" and enters their FTP credentials (provided by Adobe).
4.  **System:** Performs a "Connection Test."
5.  **Outcome:** Connection verified; User is now ready to automate distribution.
